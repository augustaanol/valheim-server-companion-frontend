# Valheim Server Companion - Frontend

Interfejs uzytkownika (Frontend) dla systemu Valheim Server Companion. Aplikacja typu Single Page Application (SPA) / Server-Side Rendering (SSR) zbudowana przy uzyciu frameworka Next.js i biblioteki React. 

Projekt sluzy do wizualizacji danych, zarzadzania serwerem oraz uzytkownikami, komunikujac sie bezposrednio z naszym API napisanym w FastAPI.

## Technologie

* Next.js (React)
* Node.js
* Tailwind CSS (lub inny system stylowania wykorzystany w projekcie)
* Axios / Fetch API (Komunikacja z backendem)

## Wymagania

* Node.js (wersja 16.x lub nowsza)
* Menedzer pakietow NPM, Yarn lub pnpm
* Uruchomiony backend (valheim-server-companion-backend)
